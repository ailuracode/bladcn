# Badge Component

## Props

- `variant`: Estilo visual (default: 'default')
- `as`: Etiqueta HTML (default: 'div')
- `...attributes` (mergeable)

## Slots

- Contenido del badge

## Ejemplo de uso

```blade
<x-badge variant="secondary">Nuevo</x-badge>
```

## Detalles

Badge visual para etiquetas, configurable por variante y etiqueta HTML.
