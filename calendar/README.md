# Calendar Component

## Props

- `value`: Valor inicial
- `wire:model`: Soporte Livewire
- `...attributes` (mergeable)

## Slots

- Contenido personalizado

## Ejemplo de uso

```blade
<x-calendar value="2026-02-21" />
```

## Detalles

Calendario interactivo, soporta selección de fecha y navegación por meses, sin subcomponentes.
